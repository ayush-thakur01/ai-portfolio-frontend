# AI Portfolio Frontend

This is the frontend of an AI-integrated portfolio system built using React and TypeScript.

## Overview

The frontend provides a modern, responsive portfolio interface with:

- Animated UI components
- Categorized skill sections
- Project showcase
- AI-powered assistant modal
- Interactive background effects

The AI assistant communicates with a FastAPI backend deployed on Render.

## Tech Stack

- React (TypeScript)
- Tailwind CSS
- Framer Motion
- Axios
- Vercel (Deployment)

## Features

- Premium UI design
- AI modal assistant
- Animated skill sections
- Project showcase with GitHub links
- Backend API integration

## Deployment

Frontend is deployed on Vercel.
